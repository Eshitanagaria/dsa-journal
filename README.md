# LeetCode Solutions

Tracking LC problems with code + actual thought process (not just final answers).

## How this works
1. New problem -> new folder: `topic/000X-problem-name/`
2. Copy `TEMPLATE.md` into that folder as `notes.md`, fill it in (metadata block at top + your approach(es))
3. Add your solution file(s) — if multiple approaches, use `brute-force.cpp`, `optimal.cpp` etc.
4. Run `python generate_readme.py` from repo root — auto-updates the table below
5. `git add . && git commit -m "..." && git push`
